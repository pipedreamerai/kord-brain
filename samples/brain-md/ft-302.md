# [[ft-302]] — 1st Pass Reject Flow Transmitter

Georg Fischer Signet 3-2537-6C-P0 paddlewheel flow transmitter. Range: 0.3–20 ft/s, 4-20 mA output. Monitors 1st-pass reject (concentrate) flow.

## Function

Paired with [[ft-301]] to calculate 1st-pass recovery: Recovery% = (Feed - Reject) / Feed × 100. At design: Feed 128 GPM, Reject 28 GPM → ~78% recovery. The PLC uses this ratio for performance trending and alarm.
