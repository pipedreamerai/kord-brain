# [[IR-2]] — PLC Input Rack

16-point, 24VDC discrete input rack. Receives dry-contact inputs from process instrumentation and feeds them to the motor start permissive logic.

## Inputs in service

- [[LSL-201]] — low-low level on [[T-101]]
- [[CV-301]] — discharge valve AUTO-position contact

Both interlocks must be satisfied for [[M-101]] to start.
